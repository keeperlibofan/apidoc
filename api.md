API
```
√ /1.0/service.register             // 用户注册的api

. /1.0/foot.recognize               // 足部识别api

. /1.0/order.create                 // 生成订单
. /1.0/answerSheet.create           // 生成问卷
. /1.0/promotionCode.draw           // 抽取优惠码
```
